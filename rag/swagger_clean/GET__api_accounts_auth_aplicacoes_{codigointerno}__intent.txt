TYPE: INTENT

SUMMARY:
Detalhar aplicação pública

DESCRIPTION:
Retorna os dados de uma aplicação pública específica pelo seu `codigointerno`.

**Acesso:** endpoint público (`AllowAny`).
**Autenticação:** não requer autenticação.
**Sessão:** não usa cookie de sessão.

**Uso principal:** consulta de uma aplicação exibida no seletor da tela de login.

**Filtro aplicado:** somente aplicações ativas e visíveis são expostas por este endpoint.
