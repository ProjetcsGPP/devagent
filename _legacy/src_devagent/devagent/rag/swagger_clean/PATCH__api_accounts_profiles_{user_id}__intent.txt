TYPE: INTENT

SUMMARY:
Atualizar parcialmente perfil de usuário

DESCRIPTION:
Atualiza parcialmente um `UserProfile` existente.

**Autenticação:** requer sessão ativa via cookie HttpOnly `gpp_session_<APP>` (ex: `gpp_session_PORTAL`). O frontend deve enviar todas as requisições com `withCredentials: true`.

**Parâmetro de rota:** `user_id` identifica o usuário dono do perfil.

**Permissões e cenários de `403 Forbidden`:**
- sem permissão para editar o perfil alvo;
- tentativa de alterar `classificacao_usuario` sem privilégio administrativo;
- tentativa de alterar `status_usuario` sem privilégio administrativo.

**Observação:** esta operação é parcial (`PATCH`), portanto apenas os campos enviados no payload serão alterados.
